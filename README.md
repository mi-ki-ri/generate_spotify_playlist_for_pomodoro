# Generate Spotify Playlist For Pomodoro

## Instalation

```sh
pip install -r requirements.txt

export SPOTIPY_CLIENT_ID=987654321
export SPOTIPY_CLIENT_SECRET=123456789
export SPOTIPY_REDIRECT_URI=http://127.0.0.1:9090

python app.py
```
